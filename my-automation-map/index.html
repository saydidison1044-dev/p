键盘快捷键 · Hammerspoon 常驻，开机自动生效

↑↓上下键变滚动
在浏览器和阅读类 App 里，上下方向键直接滚动页面，手不用碰触控板。生效范围：Safari / Chrome / Arc / Edge / Firefox / Brave / 预览(PDF) / 图书 / Claude / NotebookLM / ChatGPT / Gemini / Perplexity。写字记笔记的 App 不受影响，上下键照常移动光标。
⌘ ←⌘ →左右切换窗口
在当前所有打开的窗口之间切换，按屏幕上左右位置排序，比 ⌘Tab 更直观。
⇧⌘C窗口居中
当前窗口调成正常尺寸并居中；再按一次恢复到原来的大小和位置。（Chrome 和 Infuse 打开时会自动居中，不用手动按。）
⌥ SSession 计时器 显示/隐藏
开关 Session 番茄钟窗口，按一次显示、再按隐藏。
⌥ PSpotify 播放 / 暂停
全局播放/暂停 Spotify，不弹出窗口打断你。
⌃⌥ S剪贴板存进 Reader
复制任何文字或链接后按这个键，直接存进 Readwise Reader 收件箱。（这是 Raycast 热键，对应「存到 Reader」命令。）
\ 反斜杠键 = Esc
键盘上的「\」键被映射成 Escape，弥补物理键位。

内容生产线 · Raycast 里搜名字回车即用

共同点：打开一个已配置好的 Claude Code 窗口，素材丢进去，成品出来，中间不用管。

小红书创作
丢健康类干货素材，自动产出封面图 + 卡片图 + 正文文案，直接可发小红书。
Raycast 搜「小红书创作」
X推文创作
丢一句感悟 / 观点 / 粗想法，自动加工成 2-3 个版本的可发布推文（纯文字，不出图）。
Raycast 搜「X推文创作」
决策顾问
纠结的事（尤其恋爱相关）丢给它：信息不够它先提问，问够了给带明确结论的决策方案，自动记档案。
Raycast 搜「决策顾问」
Reader 助手
你的 Readwise Reader 阅读库接了 AI：搜索、综合回答、分类、打标签、整理，能答也能直接动手改。
Raycast 搜「Reader 助手」
读书报告
读完一本书输入书名，自动生成六模块读书报告（含联网查外部评价），投进 Reader 收件箱存档。
Raycast 搜「读书报告」
笔记库
在 Obsidian 笔记库（MyNotes）里打开 Claude Code，帮你记录、整理、归类、复盘笔记。
Raycast 搜「笔记库」
书籍提问
弹出固定顺序的读书提问模块清单，上下键选一条自动复制，粘贴到 NotebookLM 里问书用。
Raycast 搜「书籍提问」

模式开关 · 一键布置 / 收起工作环境

学习模式开关
一键三分屏：左 Claude / 中 NotebookLM / 右 Gemini，无缝铺满屏幕；已开着时再按一次就全部收起。
Raycast 搜「学习模式」
打开 Gemini 和 ChatGPT
点一次同时打开 Gemini 和 ChatGPT 窗口，再点一次全部关闭。
Raycast 搜「打开 Gemini」
专注当前窗口
开关式：按一次隐藏其他所有窗口、当前窗口居中放大；再按一次还原。
Raycast 搜「专注」
关闭模式 / 休息模式
退出 Dock 底栏里的普通应用，保留微信、v2rayN、Codex、Raycast、Finder 等常驻件。两个命令效果相同。
Raycast 搜「关闭模式」或「休息模式」
清空 Dock 运行应用
只退出 Dock 底栏的普通应用，不碰菜单栏和后台工具。（比上面两个更彻底，不留白名单。）
Raycast 搜「清空 Dock」

小工具 · 单点功能，用完即走

存到 Reader⌃⌥ S
把剪贴板里的文字 / 链接一键存进 Readwise Reader。
按快捷键，或 Raycast 搜「存到 Reader」
发剪贴板到 iPhone
抓当前剪贴板，通过 LocalSend 局域网直接推到 iPhone，不用开 AirDrop。
Raycast 搜「发剪贴板」
AI HTML网页
列出「Downloads/AI HTML网页」里 Claude 做过的所有网页，选一个用 Chrome 打开。这份对照表本身也存在那里。
Raycast 搜「AI HTML」
🔋 充电提醒（键盘+鼠标）
立刻弹一条提醒，顺手给 Brydge 键盘和 MX Anywhere 鼠标充电。（后台每周一/周四上午 10 点也会自动弹，见下方后台守护。）
Raycast 搜「充电提醒」
搜书 SingleLogin
输入书名直接打开 SingleLogin 的搜索结果页，找电子书。
Raycast 搜「搜书」
搜视频
打开常用影视搜索网站首页。
Raycast 搜「搜视频」
YouTube
用 Chrome 打开 YouTube。
Raycast 搜「YouTube」
Spotify 下一首 / 播放暂停
「Spotify Next Track」快速切下一首；「Spotify Play/Pause Hidden」播放暂停且不弹窗口（同 ⌥P 快捷键）。
Raycast 搜「Spotify」

自动定时任务 · Claude 定时跑，你什么都不用做

这些会自己运行。列在这里只是让你知道「那件事是它干的」。

Reader 每日打标每天 23:00
自动把当天新存进 Reader 的内容改成清晰标题、打上中文场景标签（搞钱变现 / 健康精力 / 认知升级…八选二）。
文件自动归类每周日 19:00
把 Downloads / 桌面 / 文稿顶层散落的文件自动归进分类文件夹。只移动、绝不删除，日志留在 .file_organizer_logs。找不到文件时先来这查日志。
Reader 周报每周日 20:00
把本周存入 Reader 的内容写成中文周报投进收件箱，开头附一行「自动化体检」——检查以上这些自动任务是否都还活着。
Drafts 每日摘要已停用
曾经每天 23 点汇总 Drafts，现已关闭，不会运行。

后台守护 · launchd 系统级常驻，开机即活

键鼠充电提醒每周一 / 周四 10:00
蓝牙键鼠电量读不到，定时弹系统通知提醒充电兜底。
NAS 影视入库确认每 3 分钟
监控夸克下载到 NAS 的影视文件，下载完弹窗让你确认片名，然后自动改成 Infuse 认得的格式入库。
夸克下载监控常驻
盯着夸克网盘的下载任务，配合上面的入库确认使用。
Dock 闲置应用自动退出常驻
长时间没用的 Dock 应用自动帮你退出，给内存腾地方。
开机自启三件套开机
v2rayN（代理）、Hammerspoon（上面所有快捷键的引擎）、美国住宅代理保活（每 15 秒检查一次连接）。快捷键全部失灵时，先检查 Hammerspoon 是否活着。
